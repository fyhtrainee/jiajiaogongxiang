# 该项目的登录地址为login/toLogin
***主要模块有：***
- 登录模块
- 注册模块
- 订单模块
- 任务模块

1. 登录模块的介绍：
> 本项目集成了shiro实现登录验证，
> 用户传入的username和pwd会包装成token传递给realm进行md5加密验证
> 后端存储的也是密文密码
2. 注册模块的介绍
> 注册时的密码经过加密存入数据库，加密方式与登录时的验证方式一致
3. 订单模块的介绍
> 订单模块有几个主要的功能1.创建订单，在这个业务设计中，
> 创建订单是需要支付费用的，但是目前没有设计订单的确认业务，
> 后续会添加。在创建订单时，创建者会得到一个四位验证码，只有输入该
> 验证码订单才可以继续。
